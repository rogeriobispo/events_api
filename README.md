## Requiriments
    * Ruby Version: 2.6.0
    * Rails 6.0.2.2
    * PostgreSQL 10.12


## Setup
    * git clone git@github.com:rogeriobispo/events_api.git
    * bundle install
    * rails db:create
    * rails db:migrate
    * rails db:seed
    * rails s
